E-Commerce Sales Performance Analysis

Project Overview

This project analyzes e-commerce sales data to identify sales trends, top-performing products, and revenue-generating categories. Using Python and data analytics libraries, the project transforms raw sales records into meaningful business insights through data cleaning, exploratory data analysis (EDA), and visualization techniques.

Objectives

  - Clean and preprocess sales data.
  - Handle missing values effectively.
  - Calculate product revenue.
  - Identify top-selling products.
  - Analyze monthly sales trends.
  - Examine category-wise revenue contribution.
  - Visualize insights using charts and graphs.

Dataset Information

The dataset contains the following columns:

 - Order ID
 - Order Date
 - Product
 - Category
 - Quantity
 - Price
 - Customer City

 A new column named Revenue is created using:

 Revenue = Quantity × Price

Project Workflow

1. Import required libraries.
2. Load CSV dataset.
3. Preview dataset using "head()".
4. Check dataset structure using "info()".
5. Identify missing values.
6. Handle missing data using mean, median, and mode.
7. Create Revenue column.
8. Perform Exploratory Data Analysis (EDA).
9. Analyze top-selling products.
10. Analyze monthly sales trends.
11. Generate visualizations.
12. Interpret business insights.

- Sales performance varied across different months.
- Some product categories contributed more revenue than others.
- Revenue distribution helped identify sales concentration and outliers.
